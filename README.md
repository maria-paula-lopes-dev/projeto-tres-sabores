# 🍦 Projeto Três Sabores

Projeto de estudo desenvolvido com **HTML5 e CSS3**, focado na criação de um layout simples e responsivo utilizando **Flexbox, organização em cards e media queries**.

## 🔗 Demonstração
👉 Acesse o projeto online:  
https://maria-paula-lopes-dev.github.io/projeto-tres-sabores/

---

## 📌 Sobre o projeto

O **Projeto Três Sabores** apresenta três sabores de sorvete organizados em **cards informativos**, contendo imagem, título e pequenas curiosidades sobre cada sabor.

A proposta do projeto foi praticar **estruturação semântica do HTML**, organização visual do layout e construção de uma página adaptável para diferentes tamanhos de tela.

---

## 🎯 Objetivos

- Criar uma página simples com **layout organizado**
- Praticar **componentes em formato de cards**
- Utilizar **Flexbox para estruturação do layout**
- Implementar **responsividade com media queries**
- Trabalhar **paleta de cores temática**

---

## 🧠 Conceitos aplicados

- Estrutura semântica do **HTML5**
- Layout com **Flexbox**
- Organização visual com **cards**
- Uso de **variáveis CSS**
- Aplicação de **pseudo-classes e pseudo-elementos**
- Responsividade com **media queries**
- Separação de estilos em arquivos CSS distintos

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  
- Media Queries  
- Google Fonts
